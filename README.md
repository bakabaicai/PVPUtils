# Old Animation

一个为 Minecraft 1.21.1 打造的开源动画模组。

## 🌟 功能特点
* **多样化防砍动画**：提供多种不同的防砍（Sword Block）动画样式，满足不同玩家的视觉偏好。
* **经典还原**：还原了 1.8 版本的“砍吃”（一边吃东西一边挥舞手臂）的动画效果。
* **自由位置调整**：支持动画在 X、Y、Z 轴上的偏移自定义。
* **核心功能开关**：格挡动画、使用动画等功能均可独立开启或关闭。
