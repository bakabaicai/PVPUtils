# ⚔️ PVPUtils

一款专为 Minecraft 1.21.11 打造的高性能 PVP 辅助工具类模组。

---

## ✨ 功能特性

### 🎮 战斗反馈 (Combat Feedback)
* **复刻1.8版本的防砍动画** > 仅限视觉效果，完美还原经典手感，并提供多种动画模式自由切换。
* **命中提示** > 在你远程或近战命中对手时，在准星周围显示类似于 FPS 游戏的命中反馈，并支持自定义命中音效。
* **目标HUD** > 实时显示当前攻击对手的生命值。  
  > *注：仅在服务器下发肉眼可见的生命值时生效；请注意部分服务器可能将其视作违规功能。*

### ⚠️ 安全预警 (Safety Alerts)
* **摔落伤害预测** > 在你将收到摔落伤害时发出提示，并在伤害足以致死时提供警告。
* **低生命值预警** > 当生命值低于设定阈值时，通过显眼的屏幕弹窗和提示音提醒你及时补充状态。

### 🏆 荣耀时刻 (Victory Moments)
* **胜利自动截图** > 获胜瞬间自动截屏并输出至系统桌面。深度适配 **Hypixel**、**MMC**、**PVPclub** 等主流服务器。
* **胜利音效** > 在你获得胜利时，播放你预设的自定义专属音效。

---

## 🛠️ 安装要求

| 环境组件 | 版本要求 |
| :--- | :--- |

| **Fabric Loader** | `>= 0.16.0` |
